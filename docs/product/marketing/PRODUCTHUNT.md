🎨 **WYSIWYG Markdown Editing**

## Pitfalls to Avoid

| Pitfall | Prevention |
|---------|------------|
| Launching too early | Test with 5 users 48hrs before |
| Ignoring comments | Block launch day, pre-write responses |
| Over-promising | Launch core value prop only |
| Vote manipulation | Organic only, never pay |
| Launch and ghost | Plan 30-day post-launch calendar |
| Wrong launch day | Tuesday/Wednesday, 12:01 AM PST |
| No hook in first 10 sec | Lead with unique value |
| Technical jargon | Plain English first |
| No social proof | Get 3-5 beta quotes beforehand |
| Arguing with critics | Stay curious: "Can you elaborate?" |
