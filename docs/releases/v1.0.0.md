# Ritemark v1.0.0

**Release Date:** 2026-01-10 (internal)

## Overview

First versioned release of Ritemark Native - a VS Code-based markdown editor with AI assistance.

## Features

### Core Editor
- TipTap-based WYSIWYG markdown editor
- Full markdown syntax support
- Auto-save with 1 second delay
- Light mode optimized theme

### AI Assistant
- AI chat sidebar (Cmd+Shift+A)
- Text rephrasing and improvement tools
- OpenAI API integration

### Document Properties
- YAML front-matter editing
- Visual property editor (text, date, tags, status)

### Task Lists
- GFM-compatible checkboxes
- Slash command `/task` for quick creation

### Block Editing
- Drag handle for reordering blocks
- Delete button on hover

### Smart Paste
- Intelligent paste from web pages
- Word/Google Docs support
- Clean HTML-to-markdown conversion

### File Icons
- Custom Lucide-based file icon theme
- Branded explorer experience

### Installer
- macOS DMG with drag-to-Applications

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon)
- Sprints completed: 01-15
