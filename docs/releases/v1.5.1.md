## What's New in v1.5.1

### Mermaid Diagram Rendering

Mermaid code blocks now render as visual diagrams directly in the editor.

- **Live diagram rendering** — fenced `mermaid` code blocks automatically render as SVG diagrams with theme-aware styling
- **Code/Diagram toggle** — switch between the rendered diagram and raw mermaid source with a single click
- **Export support** — mermaid diagrams export as embedded PNG images in both Word (.docx) and PDF
- **Broad diagram support** — flowcharts, sequence diagrams, class diagrams, state diagrams, ER diagrams, Gantt charts, and pie charts

### AI Agent Reliability

Claude and Codex plan/question workflows are now significantly more reliable.

- **Interactive questions work reliably** — Claude and Codex render inline multiple-choice questions and resume after you answer
- **Explicit plan approval** — plans are surfaced as reviewable UI instead of getting lost in chat text
- **Current plan widget** — a compact live plan summary stays visible above the input during execution
- **Better lifecycle resets** — clearing chat and starting fresh conversations clean up plan state predictably

### Extensions Marketplace

- **Install VS Code extensions** — browse and install extensions from the VS Code marketplace directly in Ritemark

### UI Polish

- **Cleaner menus** — removed Go, Terminal, and Run menus; cleaned Edit and View menus
- **Simplified New File** — file picker now shows Markdown and CSV only
- **Cmd+B fixed** — triggers Bold formatting instead of sidebar toggle
- **Debug trace logging** — optional runtime trace logging for AI agent troubleshooting (Settings → AI → Debug Trace)
