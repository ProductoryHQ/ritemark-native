# RiteMark v1.0.2-ext.1

**Release Date:** 2026-01-17

## What's New

### Spreadsheet Refresh and File Watching

This extension-only release adds the ability to refresh spreadsheet content when external files change.

#### Refresh Button
- New refresh button in the spreadsheet toolbar (circular arrow icon)
- Click to reload the current Excel or CSV file from disk
- Useful when working alongside Excel, Numbers, or other spreadsheet applications

#### Automatic File Watching
- RiteMark now detects when spreadsheet files change externally
- Blue badge indicator appears on the refresh button when changes are detected
- Stay aware of updates without constant manual checking

#### Conflict Detection
- When the file has changed externally, clicking refresh shows a confirmation dialog
- Prevents accidental data loss by warning before overwriting local view
- Clear messaging about what will happen when you refresh

### Bug Fixes

#### CSV Save Fix
- Fixed an issue where saving CSV files could fail silently
- Improved reliability of CSV write operations

## Technical Details

- Base: VS Code OSS 1.94.0
- Extension version: 1.0.2-ext.1
- Platform: macOS (Apple Silicon)
- This is an extension-only release (no app rebuild required)

## Upgrade Instructions

1. Download RiteMark.dmg from GitHub Releases
2. Open the DMG
3. Drag RiteMark to Applications (replace existing)
4. Launch RiteMark
