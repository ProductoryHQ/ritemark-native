# RiteMark v1.1.0

**Released:** 2026-01-26

## What's New

### Document Search

Ask questions about your own documents and get answers with source citations.

- **Index your workspace:** Click "Re-index" in the AI sidebar footer
- **Natural language queries:** "What did I write about project X?"
- **Source citations:** See exactly where answers come from
- **Privacy-first:** Search is local, files never leave your computer

### How to Use

1. Open a workspace with markdown files
2. Click the RiteMark AI icon in the sidebar
3. Click "Re-index" in the footer
4. Ask questions about your documents

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon)
- Vector database: Orama (local)
- Embedding model: OpenAI text-embedding-3-small

## Requirements

- macOS 11+ (Apple Silicon)
- OpenAI API key (for embeddings)

## Installation

1. Download RiteMark.dmg
2. Drag to Applications (replace existing)
3. Launch and click "Re-index" to build search index
