# RiteMark v1.0.2

**Release Date:** 2026-01-16

## What's New

### Bug Fixes

#### Git Extension Fix
- Fixed Git extension loading error that occurred when opening folders with Git repositories
- The `jschardet` module is now properly bundled with the Git extension
- This resolves the "Cannot find module 'jschardet'" error that appeared in the developer console

### Maintenance

- Rebuilt extension bundle with improved stability
- Updated webview bundle (1.4MB)

## Technical Details

- Base: VS Code OSS 1.94.0
- Extension version: 1.0.2
- Platform: macOS (Apple Silicon)
- New patch: `008-fix-jschardet-git-extension.patch`

## Upgrade Instructions

1. Download RiteMark.dmg from GitHub Releases
2. Open the DMG
3. Drag RiteMark to Applications (replace existing)
4. Launch RiteMark
