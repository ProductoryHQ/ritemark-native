# RiteMark v1.1.1

**Released:** 2026-01-30

## What's New

### Image Improvements

- **Insert images from files:** Type `/image` and select a file from your computer. Images are saved to `./images/` folder automatically.
- **Image resize handles:** Click on an image to see resize handles. Drag to resize - the actual image file is resized (keeps markdown clean).
- **Better image handling:** Fixed issues with special characters in filenames and empty paragraphs around images.

### Editor Polish

- **Stale file indicator:** When a file changes externally, a Refresh button appears in the header (matches Data Editor behavior).
- **Blockquote in bubble menu:** Quote button added to text formatting menu.
- **Cleaner bubble menu:** Table button removed (tables are still available via `/table` command).

### Under the Hood

- Shared Dialog component for consistent modals
- Improved markdown parsing for images
- Filename sanitization for cross-platform compatibility
