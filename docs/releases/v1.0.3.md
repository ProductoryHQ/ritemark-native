# RiteMark v1.0.3

**Release Date:** 2026-01-23

## What's New

### Voice Dictation (Sprint 23)

- Real-time voice dictation with bundled whisper.cpp - zero installation required
- Estonian language support with multi-language capability
- Mic toggle button in editor toolbar
- Model downloads on first use (~244MB, stored locally)
- Text appears as you speak (real-time streaming)
- Runs entirely locally - no cloud services, no data leaves your machine

### Copy as Markdown (Sprint 22)

- New "Copy as Markdown" option in Export menu
- With text selected: copies only selection as clean markdown
- Without selection: copies entire document
- Properly handles tables, lists, images, and other formatting

### Bug Fixes

- Fixed Properties dialog not scrolling when document has many properties (15+)
- Dialog content now scrolls within max-height boundary

## Technical Details

- Base: VS Code OSS 1.94.0
- Extension version: 1.0.3
- Platform: macOS (Apple Silicon)
- Bundled: whisper.cpp binary + dylibs (darwin-arm64)

## Upgrade Instructions

1. Download RiteMark.dmg from GitHub Releases
2. Open the DMG
3. Drag RiteMark to Applications (replace existing)
4. Launch RiteMark
