## LinkedIn

**Ritemark v1.3.0 is here: One app for all your documents**

We've expanded Ritemark from a markdown editor into a unified document workspace. Here's what's new:

**PDF Preview**
Open PDFs with continuous scroll, text selection, zoom controls, and page navigation. Not just viewing - you can select and copy text, making PDFs as usable as native documents.

**Word Document Viewer**
.docx files now open in Ritemark with faithful visual rendering. Fonts, colors, alignment, images, and tables all preserved. Perfect for reviewing formatted reports without switching apps.

**CSV Data Operations**
Building on our spreadsheet preview, you can now sort columns (click headers) and add rows (toolbar button). Quick data exploration without leaving your writing environment.

**Claude Code Integration**
Our Flows automation system now includes a Claude Code node, enabling agentic workflows where AI performs research/analysis tasks automatically.

**Platform Expansion**
Intel Mac support added (darwin-x64), plus automated GitHub Actions builds for faster releases across macOS and Windows.

Ritemark continues to be **privacy-first** (all processing local), **offline-capable**, and built on VS Code OSS for a familiar, powerful foundation.

Download the latest release: https://github.com/jarmo-productory/ritemark-public/releases
