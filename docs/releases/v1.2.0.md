# Ritemark v1.2.0

**Released:** 2026-02-02

## What's New

### Ritemark Flows

Visual workflow automation for AI-powered content generation.

- **New Activity Bar tab:** Flows icon in the sidebar opens the workflow editor
- **Drag-and-drop nodes:** Build workflows visually with React Flow
- **Node types:**
  - Trigger Node - Start your flow with input variables
  - LLM Node - Generate text with OpenAI GPT
  - Image Node - Create images with GPT Image 1.5
  - Save File Node - Output markdown, CSV, or images
- **Auto-layout:** Flows automatically arrange with ELKjs
- **Undo/redo:** Full history support for flow editing
- **Local storage:** Flows saved to `.ritemark/flows/` in your workspace

### Ritemark Settings

New branded settings page accessible from the Flows sidebar. Configure AI providers and preferences in one place.

### Windows Improvements

- Dictate button hidden on Windows (voice dictation is macOS-only)
- PDF export: images now properly embedded
- PDF export: unicode checkboxes render correctly
- Word export: improved line-ending compatibility

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon), Windows
- Bundle size: +2.3MB (React Flow + ELKjs)
- Feature flag: `ritemark-flows` (enabled by default)

## Requirements

- macOS 11+ (Apple Silicon) or Windows 10+
- OpenAI API key (for LLM and Image nodes)
