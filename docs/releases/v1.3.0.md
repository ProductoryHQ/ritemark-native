# Ritemark v1.3.0

**Released:** 2026-02-06

## What's New

### PDF Preview

Open any PDF file in Ritemark with a clean, document-focused viewer:

- **Continuous scroll:** All pages in one scrolling view
- **Text selection:** Select and copy text from PDFs
- **Zoom controls:** 50%, 75%, 100%, 125%, 150%, 200%, Fit Width, Fit Page
- **Page navigation:** Jump to any page with keyboard shortcuts
- **Lazy loading:** Large PDFs render pages as you scroll (performance-optimized)

### DOCX Preview

Open Microsoft Word documents (.docx) with faithful visual rendering:

- **Colors intact:** Text colors, highlights, and backgrounds match the original
- **Layout respected:** Alignment, columns, and page structure preserved
- **Images included:** Inline images render correctly
- **Tables formatted:** Cell widths, borders, and padding maintained

### CSV Editing Enhancements

Building on the existing CSV preview, v1.3.0 adds interactive data operations:

- **Column sorting:** Click any column header to sort ascending/descending/unsorted
- **Add rows:** New toolbar button to append empty rows
- **Sort-safe editing:** Cell editing works correctly while sorted (original row indices preserved)

### Claude Code Node for Flows

Use Claude Code as a node in your visual workflows:

- **Full Agent SDK integration:** Uses the official `@anthropic-ai/claude-agent-sdk` for reliable execution
- **Real-time progress:** See what Claude is doing - reading files, editing code, running commands
- **Variable interpolation:** Use `{Input Label}` or `{Node Label}` to pass data between nodes
- **Chain with other nodes:** Claude Code outputs file paths and summaries that can feed into LLM or Save nodes

### Platform Expansion

- **Intel Mac support:** Native darwin-x64 builds, no Rosetta required
- **Windows CI:** Automated Windows builds via GitHub Actions, triggered on git tags
- **Separate downloads:** Choose the right installer for your platform

## Bug Fixes

- Flows sidebar white background on VS Code light themes
- CSV editing preserves correct row indices during sort operations
- Package dependency conflicts with zod 4.x (upgraded openai to v6)

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon + Intel), Windows
- Webview bundle: ~5MB (includes react-pdf + docx-preview)
- New dependencies: `react-pdf@10.3.0`, `docx-preview@0.3.4`, `@anthropic-ai/claude-agent-sdk@0.2.29`

## Requirements

- macOS 11+ (Apple Silicon or Intel) or Windows 10+
- OpenAI API key (for LLM and Image nodes)
- Claude Code CLI installed (for Claude Code node)

## Installation

**New users:**
1. Download the DMG for your Mac architecture (or .exe for Windows)
2. macOS: Drag to Applications
3. Launch Ritemark

**Existing users:**
- Ritemark will notify you of the update, or download the new installer
