# Ritemark v1.3.0

**Released:** 2026-02-04

## What's New

### Claude Code Node for Flows

Use Claude Code as a node in your visual workflows. Claude can read files, write code, run commands, and make complex changes - all within your flow.

- **Full Agent SDK integration:** Uses the official `@anthropic-ai/claude-agent-sdk` for reliable execution
- **Real-time progress:** See what Claude is doing - reading files, editing code, running commands
- **Variable interpolation:** Use `{Input Label}` or `{Node Label}` to pass data between nodes
- **Chain with other nodes:** Claude Code outputs file paths and summaries that can feed into LLM or Save nodes

**Example workflow:** Trigger → LLM (create plan) → Claude Code (implement) → Save (output report)

### Intel Mac Support

Ritemark now builds natively for Intel Macs (darwin-x64) in addition to Apple Silicon.

- **Native Intel binary:** No Rosetta 2 required
- **Same features:** Full functionality on both architectures
- **Separate downloads:** Choose the right DMG for your Mac

### Improved Node Palette

- **"Actions" category:** Save File moved from "Output" to "Actions" - clearer that it can be chained
- **Output handles:** Save File now shows an output handle, making it obvious you can chain nodes after it
- **Better descriptions:** Clearer text explaining what each node outputs

## Bug Fixes

- **DMG notarization:** Fixed notarization workflow to sign DMG file (not just .app), resolving Gatekeeper warnings on some systems

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon + Intel), Windows
- New dependency: `@anthropic-ai/claude-agent-sdk` v0.2.29

## Requirements

- macOS 11+ (Apple Silicon or Intel) or Windows 10+
- OpenAI API key (for LLM and Image nodes)
- Claude Code CLI installed (for Claude Code node)

## Installation

**New users:**
1. Download the DMG for your Mac architecture (or .exe for Windows)
2. macOS: Drag to Applications
3. Launch Ritemark

**Existing users:**
- Ritemark will notify you of the update, or download the new installer
