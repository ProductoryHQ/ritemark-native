# Ritemark v1.3.1

**Released:** 2026-02-08

This is a hotfix release that resolves an issue introduced in v1.3.0 where the Settings page was inaccessible, preventing users from configuring API keys needed for AI features.

## Fixes

- **Settings page restored** — The Settings page was accidentally removed in v1.3.0, which meant users could not configure OpenAI or Gemini API keys. All AI features (Flows, Chat) are now fully accessible again.
- **Removed false "corrupt installation" warning** — Windows users no longer see a misleading warning about corrupted installations on startup.
- **Consistent Light theme on launch** — New windows now default to the Light theme instead of briefly flashing a dark background.
- **Welcome page logo** — The Ritemark logo now displays correctly on the Welcome tab across all platforms.

## Installation

**New users:**
1. Download the DMG for your Mac architecture
2. Drag to Applications
3. Launch Ritemark

**Existing users:**
- Download and install the new version over your existing installation
- Your settings and files are preserved

## Requirements

- macOS 11+ (Apple Silicon or Intel) or Windows 10+
