# Ritemark v1.4.0

**Released:** 2026-02-14

## What's New

### Claude Code Agent in AI Sidebar

Use Claude Code directly from the Ritemark AI sidebar for autonomous file operations:

- **Persistent agent sessions:** Multi-turn conversations with fast follow-ups (~2-3s vs ~8s)
- **Real-time activity feed:** See what Claude is doing — reading files, editing code, running commands
- **Image paste support:** Cmd+V screenshots into chat for visual context
- **Clickable file paths:** Code paths in responses open files when clicked
- **Active file context:** Agent automatically sees your currently focused file
- **Excluded folders:** Configure which folders the agent cannot access (Settings)
- **Model selector:** Choose between Claude Sonnet, Opus, or Haiku

**Note:** This feature is experimental and requires enabling in Settings (Ritemark Features > Agentic AI Assistant). Requires Claude Code CLI installed.

### Refreshed Visual Design

Ritemark gets a cleaner, writer-focused aesthetic:

- **Sofia Sans font:** Professional UI typography throughout the app
- **Refined color theme:** Softer grays, indigo accents, improved contrast
- **Horizontal sidebar tabs:** Activity bar tabs moved to top of sidebar (cleaner layout)
- **Updated icons:** Harmonized icon set across the interface
- **Smaller breadcrumbs:** More space for your content

### Improved CSV Editing

Excel-like cell editing experience:

- **Full-text cell editing:** See entire cell contents while editing (not truncated)
- **Active cell focus:** Clear Excel-like border on selected cell
- **Add/rename/delete columns:** Column operations via header interactions
- **Copy-paste support:** Cmd+C/V works on any selected cell
- **Improved keyboard navigation:** Enter commits, Esc cancels, Tab moves cell

### Export Improvements

- **Fixed PDF images:** Images now render correctly in PDF exports
- **Fixed Word images:** Proper aspect ratio for embedded images
- **Better PDF headings:** Consistent spacing and orphan protection
- **Cleaner export UI:** Simplified menu (removed confusing template choices)

### Bug Fixes

- **Cmd+B conflict resolved:** Bold formatting no longer toggles the sidebar
- **Terminal auto-open fixed:** Terminal only opens on startup if none exists
- **Word code blocks:** Properly formatted in exports
- **Toolbar refresh:** Spreadsheet toolbar updates correctly after operations

## Technical Details

- Base: VS Code OSS 1.94.0
- Platform: macOS (Apple Silicon + Intel), Windows
- New dependencies: Sofia Sans font bundle
- New patches: 015-021 (GUI customization)

## Requirements

- macOS 11+ (Apple Silicon or Intel) or Windows 10+
- OpenAI API key (for LLM, Image nodes, and Ritemark Agent)
- Claude Code CLI (for Claude Code agent in sidebar) — optional

## Installation

**New users:**
1. Download the DMG for your Mac architecture (or .exe for Windows)
2. macOS: Drag to Applications
3. Launch Ritemark

**Existing users:**
- Ritemark will notify you of the update, or download the new installer
- Your settings, documents, and Flows are preserved
