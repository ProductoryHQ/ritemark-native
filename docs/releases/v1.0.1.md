# Ritemark v1.0.1

**Release Date:** 2026-01-11

## What's New

### Document Header & Export (Sprint 18)

A new sticky header at the top of your documents provides quick access to:

-   **Properties** - View and edit document metadata (YAML front-matter)
    
-   **Export** - Export your documents to other formats
    

#### PDF Export

-   Export any markdown document to PDF
    
-   Clean formatting with Arial font
    
-   Supports headings, lists, code blocks, and tables
    

#### Word Export

-   Export to Microsoft Word (.docx)
    
-   Full formatting preserved
    
-   Arial font throughout
    

### CSV File Preview (Sprint 17)

Open CSV files directly in Ritemark with a spreadsheet-style view:

-   **Inline Editing** - Click any cell to edit, changes auto-save
    
-   **Row Numbers** - Excel-style row numbering
    
-   **Virtual Scrolling** - Handles large files (up to 10,000 rows)
    
-   **Large File Warning** - Automatic warning for files over 5MB
    

### Auto-Update Notifications (Sprint 16)

Ritemark now checks for updates automatically:

-   Checks GitHub Releases on startup (10 second delay)
    
-   Shows notification when new version available
    
-   Options: "Install Now", "Later", "Don't Show Again"
    
-   Respects user preferences (can be disabled in settings)
    

## Technical Details

-   Base: VS Code OSS 1.94.0
    
-   Extension version: 1.0.1
    
-   Platform: macOS (Apple Silicon)
    

## Known Issues

-   Excel file preview not yet supported (planned for future release)
    

## Upgrade Instructions

1.  Download the DMG from GitHub Releases
    
2.  Open the DMG
    
3.  Drag Ritemark to Applications (replace existing)
    
4.  Launch Ritemark
