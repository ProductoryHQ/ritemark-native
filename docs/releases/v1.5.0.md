# Ritemark v1.5.0

**Status:** In progress  
**Target release:** 2026-XX-XX

Ritemark v1.5.0 is the next full release and combines a major VS Code foundation refresh with a large reliability pass for updates and external AI agent setup.

## Planned Scope

- VS Code upstream foundation update to `1.109.5` from **Sprint 41**
- **Sprint 42: Unified Update Platform**
  - compatibility-aware update feed and resolver
  - Settings `Update Center`
  - better extension/full-update handling
  - clearer component readiness for voice model, Claude, and Codex
- Claude and Codex bootstrap/auth hardening
  - install / repair / sign-in states in Settings and sidebar
  - clearer account surfaces for `Claude.ai` and `ChatGPT`
  - fixed Codex screenshot/image delivery
  - removed misleading per-image token estimates in Codex chat
