# Ritemark v1.5.0

**Released:** 2026-03-15

## Highlights

Ritemark v1.5.0 is a foundation-and-reliability release. It upgrades the VS Code base to `1.109.5`, adds a proper Update Center, redesigns the welcome experience, and makes AI agent setup much more reliable -- especially on Windows.

## What's New

### VS Code Foundation Update (Sprint 41)

- Updated VS Code base from `1.94.0` to `1.109.5`
- Revalidated the Ritemark patch stack against the new upstream
- Fixed upgrade regressions found during smoke testing
- Improved stability and performance

### Unified Update Platform (Sprint 42)

- New Settings **Update Center** with manual check, install/download, skip, pause, resume, and restart-required handling
- Canonical update feed generation and compatibility-aware update resolution
- Full-release vs extension-update handling is now compatibility-aware instead of `latest`-only
- Component readiness cards for voice model, Claude, and Codex

### Welcome Page and Onboarding (Sprint 43)

- New branded Ritemark welcome page with "Write Smarter. Think Faster." hero
- Launch check panel showing Claude, ChatGPT, Git, and Node status at a glance
- Quick-action buttons: New document, New table, New flow, Open folder
- `Recent files` stays as a first-class shortcut for returning users

### AI Agent Setup Improvements

- Dedicated `Claude Account` and `ChatGPT Account` sections in Settings
- Claude and Codex setup states visible in both Settings page and AI sidebar
- Claude Account section shows auth method, CLI version, and billing source
- ChatGPT Account section shows email, plan, and connection status
- Claude/Codex setup recovery surfaces in the AI sidebar

### Windows AI Bootstrap Hardening (Sprint 44)

**Fixed: Claude and Codex detection on Windows**

- Claude and Codex binaries are now detected reliably on Windows. Previous versions could fail with "spawn ENOENT" or "spawn EINVAL" errors because npm installs shell script shims alongside `.cmd` wrappers, and Node.js cannot spawn the shell scripts.
- Ritemark now filters for `.exe` and `.cmd` files when searching for binaries, skipping unusable shims.

**Fixed: Claude chat "spawn EINVAL"**

- Sending a message to Claude in the AI sidebar could fail with "spawn EINVAL" on Windows. This happened because the Claude SDK runs `node <path>`, but Ritemark was passing `claude.cmd` (a batch wrapper) instead of the underlying JavaScript entry point. Ritemark now resolves `.cmd` wrappers to their actual `cli.js` path.

**Fixed: Claude install file lock**

- Clicking "Install Claude" or "Repair Claude" multiple times could cause a file lock error. Ritemark now prevents concurrent installs and clears stale download files before retrying.

**Improved: Environment checks on Windows**

- Ritemark checks for Git, PowerShell, and Node.js before attempting Claude or Codex setup, and shows which prerequisite is missing with a clear recovery action.
- Install and sign-in buttons are disabled when prerequisites are missing, preventing confusing error messages.

## Bug Fixes

- Fixed Codex approval dialog -- 4 bugs blocking all user approvals
- Fixed Codex screenshot/image attachments now reaching the agent correctly
- Removed misleading per-image token estimates from Codex chat attachments
- CSV editor column headers no longer block sorting; destructive column actions now live behind a dedicated header action menu
- Removed debug console.log from Codex approval routing

## Improvements

- New [AI Agents](../user/features/ai-agents.md) guide covering all three agents, detection, prerequisites, and troubleshooting
- Windows CI workflow restored and hardened with deterministic installs and build log upload on failure

## Requirements

- macOS 11+ (Apple Silicon or Intel) or Windows 10+
- OpenAI API key (for LLM, Image nodes, and Ritemark Agent)
- Claude Code CLI (for Claude Code agent in sidebar) -- optional
- Codex CLI (for Codex agent in sidebar) -- optional

## Installation

**New users:**
1. Download the installer for your platform
2. macOS: Drag to Applications / Windows: Run the installer
3. Launch Ritemark

**Existing users:**
- Ritemark will notify you of the update, or download the new installer
- Your settings, documents, and Flows are preserved
