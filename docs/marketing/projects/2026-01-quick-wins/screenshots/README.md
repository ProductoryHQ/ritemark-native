# Screenshot Capture Instructions

Use these example files to capture screenshots for the landing page.

---

## Screenshots Needed

| Screenshot | Example File | What to Show |
|------------|--------------|--------------|
| CSV Editing | `example-data.csv` | Spreadsheet view, cell being edited |
| Excel Preview | any `.xlsx` file | Sheet tabs + "Open in Excel" button |

---

## Setup

1. Open Ritemark
2. Dark mode preferred (matches landing page)
3. Window size ~1400×900
4. Hide unnecessary panels

---

## Capture 1: CSV Editing

**File:** `example-data.csv` (in this folder)

**Show:**
- CSV file open in spreadsheet view
- One cell actively being edited (cursor visible)
- Clean data rows visible

**Focus:** The inline editing capability

---

## Capture 2: Excel Preview

**File:** Any `.xlsx` with multiple sheets

**Show:**
- Excel file open in preview mode
- Sheet tabs visible at bottom
- Spreadsheet toolbar with **"Open in Excel"** or **"Open in Numbers"** button clearly visible

**Focus:** The one-click button to open in native app
