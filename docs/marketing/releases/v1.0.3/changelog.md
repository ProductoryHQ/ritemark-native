## [1.0.3] - 2026-01-23

### Added
- Voice dictation with bundled whisper.cpp - real-time speech-to-text running entirely locally
- Estonian language support with 50+ additional languages available
- Mic toggle button in editor toolbar for quick dictation access
- Whisper model downloads on first use (~244MB, stored locally)
- Real-time streaming text output during dictation
- "Copy as Markdown" option in Export menu
- Selection-aware copy: copies only selected text or entire document as clean markdown

### Fixed
- Properties dialog now scrolls when document has many frontmatter properties (15+)
- Dialog content respects max-height boundary
