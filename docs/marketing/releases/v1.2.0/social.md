## LinkedIn

Ritemark v1.2.0 introduces Flows - a visual workflow automation system for AI-powered content generation.

Instead of running prompts one at a time, you can now build reusable workflows by connecting nodes: Trigger -> LLM -> Image -> Save File. Execute a flow to generate blog posts, create matching images, and save everything to your workspace automatically.

The visual editor uses drag-and-drop with automatic layout. Undo/redo support. All flows stored locally in your workspace.

This release also includes Windows polish: better PDF and Word export, and platform-appropriate UI (dictate button now macOS-only).

Still local-first. Still privacy-focused. Your files and flows stay on your machine.

Download: https://ritemark.app
