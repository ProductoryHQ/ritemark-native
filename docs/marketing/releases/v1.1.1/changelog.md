## [1.1.1] - 2026-01-30

### Added
- **Insert images from files** - Type `/image` to select and insert images from your computer
- **Image resize handles** - Click any image to resize with drag handles (resizes actual file)
- **Stale file indicator** - Refresh button appears when external changes detected
- **Blockquote in bubble menu** - Quick access to quote formatting in text selection menu

### Changed
- Table button removed from bubble menu (still available via `/table` command)
- Shared Dialog component for consistent modal styling

### Fixed
- Special characters in image filenames now handled correctly
- Empty paragraphs around images cleaned up
- Cross-platform filename sanitization
