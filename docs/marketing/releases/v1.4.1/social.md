## LinkedIn

### Announcement Post
```
Ritemark v1.4.1: Multi-Agent AI for Writers and Developers

The AI sidebar in Ritemark now supports three agents: Claude Code (Anthropic), OpenAI Codex (new), and the built-in Ritemark Agent. Pick the right tool for the task and switch with a dropdown.

What else is new:

Plan Mode -- The AI proposes a plan before executing. You approve or reject. No more surprise file changes.

Interactive Questions -- When the AI needs clarification, it asks you directly with structured options instead of guessing or stalling.

File Context -- Right-click any file in the Explorer and select "Send to AI Chat" to give the agent context. The active file is attached automatically.

Context Window Protection -- A usage bar shows how much of the model's context window you have used. Warnings at 70%, clear error handling at overflow. No more silent failures in long conversations.

Ritemark remains local-first. Your files stay on your machine. No cloud sync, no account required.

Download: github.com/jarmo-productory/ritemark-public/releases
```
