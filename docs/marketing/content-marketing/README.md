# Content Marketing

Central hub for RiteMark's content marketing efforts.

## Structure

```
content-marketing/
├── README.md           # This file
├── blog/               # Blog strategy and content
│   ├── STRATEGY.md     # Living strategy document (review quarterly)
│   ├── README.md       # Blog project overview
│   └── posts/          # Draft posts (when created)
└── stats/              # Performance tracking
    └── README.md       # How we track metrics
```

## Key Documents

| Document | Purpose | Review Cadence |
|----------|---------|----------------|
| `blog/STRATEGY.md` | Blog content strategy | Quarterly |
| `stats/` | Performance metrics | Monthly |

## Workflow

1. **Strategy** guides what content we create
2. **Posts** are drafted in `blog/posts/` (ET first, then EN)
3. **Stats** inform strategy updates
