## Quick Reference

### What Ritemark Opens

| File Type | Mode | What You Can Do |
|-----------|------|-----------------|
| `.md`, `.markdown` | Edit | Full WYSIWYG editing |
| `.csv` | Edit | Inline cell editing |
| `.xlsx`, `.xls` | Preview | View and open in Excel/Numbers |

### Key Shortcuts

| Action | Shortcut |
|--------|----------|
| Bold | Cmd+B |
| Italic | Cmd+I |
| Link | Cmd+K |
| Slash commands | Type `/` |

[Full keyboard shortcuts →](keyboard-shortcuts.md)

### AI Tools

| Tool | What It Does |
|------|--------------|
| Rephrase | Rewrite selected text (shorter, formal, etc.) |
| Find & Replace | Replace all occurrences |
| Insert | Add text at specific position |
